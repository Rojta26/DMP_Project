# DMP_Project
## Requirements
To run this experiment, you need Python installed along with the libraries listed in `requirements.txt`:
Install it by typing next in python terminal:
pip install -r requirements.txt

## How to Run
1. Make sure that the Stack Overflow CSV files (so_2021.csv to so_2025.csv) are in the 'data/raw' folder.
2. Run the script:
   python src/analysis_script.py
3. The results will be generated in the 'results/' folder.
